# Shoe App

A fun project using my flutter skills to build an e-commerce app.

### Features
 - Login
 - Register
 - Add/Delete shoes into cart
 - Change username
 - View different shoes
 - Clear cart
 - Used Firebase to store data 
 - Compare prices from StockX ans GOAT
